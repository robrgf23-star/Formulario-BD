# Formulario con Acceso a Base de Datos

Sistema web completo que permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre una base de datos de usuarios.

## Características

- ✅ Formulario responsive para agregar/editar usuarios
- ✅ Búsqueda y filtrado de usuarios
- ✅ Operaciones CRUD completas
- ✅ Interfaz moderna y amigable
- ✅ Código modular y bien estructurado
